# 🐍 Python_Mini Projects 

## 📌 Overview
This repository contains multiple beginner-to-intermediate Python projects that demonstrate programming fundamentals, problem solving, and Object-Oriented Programming (OOP).

## 📂 Included Projects
- 🎯 Secret Word Game
- 🔤 Translator Program
- 🔐 Password Generator
- ✅ To-Do List App
- 🎮 Tic Tac Toe Game (OOP)
- 🏫 School Management System (OOP)
